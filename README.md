# RestrictedNavigationBar

A Java-based navigation component that restricts access to specific menu items based on user permissions or roles.

## Features
* **Role-Based Access**: Dynamically enable or disable navigation buttons based on user login.
* **Custom Styling**: Easily customizable UI elements for the navigation bar.
* **Easy Integration**: Can be integrated into existing Java Swing or JavaFX applications.
